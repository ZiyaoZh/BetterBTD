# BetterBTD

BetterBTD 是一个面向《气球塔防 6》玩家的 Windows 自动化工具。它把游戏窗口截图、界面识别、键鼠模拟、脚本编辑、脚本库管理和自动任务整合到同一个桌面程序里，目标不是“写一堆零散脚本”，而是给玩家一套可重复、可维护、可导入导出的自动化工作流。

当前仓库里的实现重点已经覆盖这些方向：

- 游戏窗口捕获与截图测试
- 基于画面识别的关卡内状态读取
- 可视化脚本编辑器
- 托管脚本库与脚本 ID 管理
- 自动任务运行器
- 回合、英雄、模范相关计算工具
- 自定义按键绑定与两种键鼠模拟模式

## 项目定位

BetterBTD 更适合下面这几类场景：

- 你想把一套稳定打法保存成脚本，重复运行
- 你想维护自己的脚本库，并按地图、难度、模式、英雄、标签管理
- 你想给收集活动、金气球、黑框、循环刷关、奥德赛配置自动任务
- 你需要一套图形界面工具，而不是完全手写脚本文本

它不是一个“只会录制点击”的录屏回放器。它更像一套围绕 BTD6 组织起来的自动化平台：脚本有元数据，自动任务有绑定关系，运行时有日志和步骤状态，导入导出也围绕这些概念设计。

## 你能在程序里做什么

- 在“开始”页启动截图器、测试捕获效果、手动指定目标窗口
- 在“自动任务”页配置不同任务模式，并查看任务运行时状态
- 在“脚本编辑器”里搭建放塔、升级、等钱、等回合、发下一轮、技能释放等指令序列
- 在“我的脚本”里导入、导出、筛选、编辑、运行和删除脚本
- 在“工具”页计算回合收益、英雄等级、模范度和模范数值
- 在“设置”里切换语言、主题、输入模拟方式，并调整游戏热键映射

## 推荐使用方式

对普通用户，最稳定的日常流程是：

1. 先手动启动 BTD6，并把游戏窗口保持在 16:9。
2. 打开 BetterBTD，在“开始”页启动截图器，必要时先做一次截图测试。
3. 到“设置”确认游戏热键和 BetterBTD 中的按键绑定一致。
4. 在“我的脚本”导入现成脚本，或在“脚本编辑器”新建并调试脚本。
5. 需要单次执行时直接运行脚本，需要批量循环时再去“自动任务”页配置任务。

## 使用建议

- 推荐系统：Windows 10/11 x64
- 推荐分辨率：16:9，优先 1920×1080
- 推荐先手动启动游戏，再让 BetterBTD 接管截图和输入
- 推荐先跑通单个脚本，再配置自动任务
- 推荐把游戏热键保持为默认，或者在 BetterBTD 的按键绑定页同步改掉

## 当前版本里最值得注意的边界

- 稳定主路径是“手动启动游戏 + BetterBTD 启动截图器/手动选窗”。开始页下半部分的联动启动相关选项目前更适合作为预留界面看待，不建议作为主要工作流依赖。
- 脚本坐标体系以 1920×1080 为参考基准，非 16:9 窗口可能出现坐标偏移。
- 自动任务与脚本执行都按“同一时刻只跑一个任务”设计，不支持并发执行多个自动流程。
- 奥德赛自动任务当前要求你先手动进入奥德赛流程，程序再接管后续每一关的脚本运行。
- 如果你启用硬件键鼠模拟，需要系统里已经安装对应驱动；否则程序会退回普通模拟模式。

## 本地数据位置

BetterBTD 会把不同类型的数据分开保存：

- 程序配置：`%LocalAppData%\BetterBTD\appsettings.json`
- 托管脚本库：程序目录下的 `User\MyScripts\`
- 脚本文件主体：`User\MyScripts\Assets\`
- 自动任务绑定文件：`User\MyScripts\Bindings\`

如果你在迁移环境，至少要同时关注“配置文件”和“User\MyScripts”这两块数据。
