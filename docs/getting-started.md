# BetterBTD 快速上手

这份文档只讲“第一次把 BetterBTD 跑起来并开始稳定使用”的步骤，不讨论代码结构，也不讨论脚本格式细节。

## 适用环境

- Windows 10 或 Windows 11
- 64 位系统
- BTD6 已正常安装并可进入游戏
- 建议使用 16:9 窗口比例
- 建议使用 1920×1080，至少先在这个比例下完成首次配置

如果你是从源码运行而不是直接使用成品程序，还需要本机具备对应的 .NET 运行环境。

## 第一次启动前先确认

- 游戏热键尽量保持默认
- 游戏窗口标题应保持常见名称，或者准备在 BetterBTD 里手动选窗
- 如果你准备使用硬件键鼠模拟，先确认相关驱动已经安装

## 第一次启动 BetterBTD 时建议这样做

1. 先打开 BTD6。
2. 打开 BetterBTD，先进入“设置”页，把界面语言改成你习惯的语言。
3. 在“设置”里检查“游戏语言”和你实际游戏界面语言一致。
4. 如果你改过 BTD6 热键，立即打开按键绑定窗口，把 BetterBTD 里的按键也改成一致。
5. 返回“开始”页，先不要急着跑脚本，先完成截图器配置。

## 开始页应该怎么用

开始页对稳定性最重要的是上半部分的截图器区域。

建议按下面顺序操作：

1. 先看“截图模式”，优先使用 `WindowsGraphicsCapture`。
2. 点击“测试图像捕获”，确认能抓到正确画面。
3. 如果程序没找到游戏窗口，使用“手动选择窗口”。
4. 截图正常后，再点击启动截图器。
5. 如果你是 Windows 11 且截图异常，可以尝试开启 Win11 兼容修复，再重启游戏验证。

## 对开始页里各项功能的实际建议

- `截图模式`
  建议优先用 `WindowsGraphicsCapture`。这是当前版本最适合作为默认值的模式。
- `测试图像捕获`
  很值得先做。它能在你写脚本前先排除抓图失败的问题。
- `手动选择窗口`
  当游戏标题不标准、存在多开、或程序找错窗口时很有用。
- `Win11 兼容修复`
  只在你确实遇到截图问题时再启用。
- `联动启动、安装路径、启动参数、自动进游戏、CMD 启动、记录游戏时间`
  当前版本不建议把它们当成主工作流的一部分。稳定路线仍然是手动启动游戏，再让 BetterBTD 接管截图和后续自动化。

## 输入模拟模式怎么选

设置页里有两种输入模拟方式。

- 普通模拟
  开箱即用，不需要额外驱动。适合绝大多数用户先用它把流程跑通。
- 硬件模拟
  需要额外驱动支持。适合普通模拟被安全软件、游戏前台状态或系统环境影响时再尝试。

如果你切到硬件模拟但驱动并不存在，程序会自动退回普通模拟，所以看到状态提示时要留意它是否真的可用。

## 按键绑定必须检查什么

BetterBTD 的执行逻辑依赖“它以为的按键”和“你游戏里真正的按键”一致。

至少要检查这些类别：

- 防御塔放置热键
- 英雄热键
- 升级三路热键
- 切换目标热键
- 卖出热键
- 技能编号热键
- 英雄物品热键
- 快进与发下一轮热键

如果这些不一致，脚本往往不是“偶尔失误”，而是会系统性跑偏。

## 第一次真正执行前的推荐流程

1. 先在“我的脚本”导入一个现成脚本。
2. 进入脚本详情，确认地图、难度、模式、英雄信息都对。
3. 直接从脚本库运行一次，观察运行时窗口。
4. 单脚本跑通后，再去配置自动任务。

不要一上来就先配收集、黑框或奥德赛。那样出问题时你很难判断是截图、热键、脚本还是自动任务导航造成的。

## 稳定性建议

- 优先使用 16:9 游戏窗口
- 第一次调试尽量不要切后台
- 调试脚本时先用简单地图和简单模式
- 有坐标相关指令时，尽量用编辑器里的坐标选择功能取点
- 先确认单脚本稳定，再导出、分发或绑定到自动任务

## 本地数据保存在哪里

- 程序配置：`%LocalAppData%\BetterBTD\appsettings.json`
- 脚本库根目录：程序目录下 `User\MyScripts\`
- 托管脚本文件：`User\MyScripts\Assets\`
- 自动任务绑定文件：`User\MyScripts\Bindings\`

如果你重装系统、换电脑或换程序目录，这几个位置是最重要的数据来源。

## 常见问题

### 程序找不到游戏窗口

- 先确认游戏已经启动
- 再用“手动选择窗口”
- 如果你是平台特定版本，确认窗口标题没有变化

### 截图测试能开，但脚本还是跑偏

- 先检查游戏热键是否和 BetterBTD 一致
- 再检查游戏窗口是否为 16:9
- 最后检查脚本本身的地图、模式和坐标是否对应当前局面

### 自动任务一启动就失败

- 先回到单脚本运行确认脚本本身可用
- 再确认该自动任务需要的脚本绑定文件或脚本 ID 已经填好
- 奥德赛模式还要额外确认你已经手动进入奥德赛流程

### F10 没反应

- F10 只在脚本运行窗口或自动任务运行窗口打开后才会注册
- 如果系统里其他软件抢占了 F10，BetterBTD 会注册失败
- 这时可以直接用窗口里的开始/停止按钮
